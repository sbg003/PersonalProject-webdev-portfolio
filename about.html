Hello World,

My name is Sarah Bennett, and I am a self-taught Junior Web Developer seeking entry-level employment doing front-end web development. I have experience building static and dynamic responsive web sites using html, Tailwind and Bootstrap CSS frameworks, React Javascript framework, and Wordpress CMS. Some of these projects are featured on the Projects tab above.

I have 15 years experience previously as a Molecular Biologist/Project Manager, mostly working in CRO environments, performing oncology research for clinical trials. 30% of my role as a scientist was spent performing UAT testing for software validations, for both house-developed data analysis software and off-the-shelf robotics software. The other 70% of my role consisted of learning new lab technologies and developing/validating new diagnostic assays targeting new cancer biomarkers.

In 2022, I chose to take time off to be a stay at home parent. During this time off I learned front-end web development skills and practiced these skills by creating websites for my entrepreneur friends. As I go back to work, I'm seeking web developer roles that are both challenging and offer work-life balance.

Consistently, in different roles over the last 15 years, I've been given feedback that I'm an excellent troubleshooter and communicator, patient mentor, and have a high morale. I value integrity, transparency, humility, and quality. I'm looking forward to fitting into an environment that shares similar values.
